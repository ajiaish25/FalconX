## How to Use in Confluence

1. **Create a new page** in Confluence.
2. **Switch to Markdown** (if available) or paste the content; Confluence will convert markdown on paste in most versions.
3. **Add color/panels** where desired:
   - `{info}` for helpful tips
   - `{warning}` for cautions
   - `{panel:title=...|borderColor=#3572b0}` for highlighted sections
4. **Tables** paste directly as Confluence tables.
5. **Code blocks** paste as Confluence code macros.
6. **ASCII diagrams** can be pasted as preformatted text, or recreated in Draw.io/Gliffy for visual diagrams.
